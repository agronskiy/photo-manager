ОПИСАНИЕ ПРОЦЕДУРЫ УСТАНОВКИ ПРОГРАММЫ ФотоМенеджер (начиная с версии 1.1.0) (с) А. Гронский, 2008

ТРЕБОВАНИЯ:

	1. БД PostgreSQL 8.3 (или выше) с поисковым движком TSearch (последний обычно
	входит в дистрибутивы PostgreSQL указанных версий и рекомендуется к установке
	в случае отсутствия такового. Отсутствие движка TSearch не приведет
	к сбою программы, однако повлечет замедление работы при поиске по фотобазе).

	2. Необходимые базы данных и таблицы могут быть созданы путем выполнения SQL-запроса,
	находящегося в файле docs/pm_database.sql

	3. Установленный модуль tablefunc для PostgreSQL (может быть установлен с помощью docs/tablefunc.sql)

ОБЩИЕ ДЕЙСТВИЯ ДЛЯ ВСЕХ ПЛАТФОРМ:

	1. Распакуйте архив в нужное место

	2. После запуска программы задайте настройки БД фототеки в меню "Опции->
	->Настройки БД и директории данных" (если Вам не известны настройки, запросите их у Вашего
	системного администратора)

УСТАНОВКА НА ПЛАТФОРМЕ Windows:

	Для установки на платформе Windows (XP, Vista) запустите файл vcredist_x86.exe (папка "Дополнительно"), после чего запуск приложения
	осуществляется с помощью файлы сценариев pc.bat

	В случае использования более низких версий ОС Windows возможны проблеме при загрузке драйвера
	базы данных, см. раздел "Проблемы при запуске"

УСТАНОВКА НА ПЛАТФОРМЕ Linux/X11:

	Запуск приложения осуществляется с помощью скрипта с расширением .sh (а НЕ самого исполняемого
	файла).

УСТАНОВКА НА ПЛАТФОРМЕ Mac OS:

	В разработке.

ПРОБЛЕМЫ ПРИ УСТАНОВКЕ/ЗАПУСКЕ:

	1. Сообщение "driver not loaded" при запуске приложения означает, что не загружен драйвер
	базы данных. Рекомендуется проверить файл sqldrivers/qsqlpsql4.dll (для Windows) или
	sqldrivers/libqsqlpsql.so (для X11) на наличие незагруженных модулей (программа DependencyWalker
	для Windows или утилита ldd для X11).

	2. Сообщение об ошибке подключения к БД "connection pointer is NULL" означает, что неверно заданы
	настройки подключения к БД (задайте настройки в меню "Опции->Настройки БД.." и снова подключитесь
	к БД в "Опции->Подключиться к БД"), либо БД не запущена (свяжитесь с Вашим системным администратором).
	В частности, такая ошибка неизбежна при первом запуске программы, т.к. программа делает попытку
	подключения к БД с незаданными настройками.

ПРОЦЕДУРА СБОРКИ ПРОГРАММЫ ИЗ ИСХОДНЫХ КОДОВ

	Необходимо:
	1. Qt 4.3 и выше

	2. Компилятор GCC

	3. Распакуйте пакет с исходными кодами в любую папку, после чего выполните команды:
		% qmake
		% make

	4. Собранный исполняемый файл поместите в корень дистрибутива (win_template/x11_template)
